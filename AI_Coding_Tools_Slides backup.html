Phần 1: Nền tảng (Foundations)

3 Công cụ AI là gì?

Trong hệ sinh thái công nghệ, ba trợ lý AI thuộc dòng CLI (Command Line Interface: giao diện sử dụng văn bản để điều khiển máy tính thay vì dùng chuột chọn/click) đang nổi bật nhất là Claude Code, Gemini CLI và AntiGravity.

Cả ba công cụ này đóng vai trò như "Trợ lý ảo chuyên sâu", giải phóng lập trình viên (Developer) và nhà phân tích dữ liệu (DA) khỏi việc gõ lệnh lặp đi lặp lại. Chúng tự động sinh đoạn mã (code), quản lý tệp và truy xuất dữ liệu hệ thống hiệu quả — tuy nhiên triết lý và độ sâu bộ nhớ của chúng lại rất khác biệt.

Phần 1: Nền tảng (Foundations)

Bản chất 3 công cụ

1. Claude Code

Bản chất: Agent tự hành (Autonomous Agent: Thực thể AI chạy ngầm có khả năng tự động lập ra kế hoạch đa bước và tự phân tích, không cần chờ bạn cầm tay chỉ việc).

Điểm mạnh: Lập luận logic cực tốt.

Điểm yếu: Bộ nhớ "cá vàng" — phụ thuộc 100% vào việc team tự phải cập nhật cuốn sổ nội quy chung mang tên CLAUDE.md.

2. Gemini CLI

Bản chất: Trợ lý xử lý kịch bản, hoạt động theo mô hình nhanh (Just-in-Time: gọi là làm, làm tức thời rồi sẽ làm trống bộ nhớ tốn kém).

Điểm mạnh: Tốc độ triển khai cực nhanh, kết nối sâu rộng cực đỉnh với Google Cloud API.

Điểm yếu: Hoàn toàn vắng bóng hệ thống lưu giữ trí nhớ dài hạn (Long-term memory).

3. AntiGravity

Bản chất: Hệ giao diện Lai (Hybrid: Có cả bảng chọn chuột dễ nhìn Sidebar UI, vừa có trí tuệ Agent thao túng sức mạnh ẩn backend).

Điểm mạnh: Nhớ thói quen tự sửa lỗi qua Knowledge Items (KIs), xuất tài liệu đồ họa cao cấp (Artifacts), và phái điệp viên ngầm lướt web tự động (Browser Subagent).

Phần 3: Gỡ rối kiến thức

Claude Code vs Code Assist

⚠️ Hãy phân định rạch ròi: Hệ thống đại diện dòng lệnh (Terminal Agent) khác xa so với loại Tool trí óc gắn trên trình gõ code của IDE.

Đặc Điểm Quan Trọng Dòng Agent Tự Hành (Claude Code / AntiGravity) AI Tích Hợp Gõ Code (Gemini Code Assist)
Phương thức tương tác Khoán việc đầu cuối (End-to-End: truyền đề bài và rảnh tay chờ nhận file kết quả tự sửa hoàn chỉnh). Phản xạ nhạy ngay lập tức (Tự động điền code hoặc giải đáp khi con người rê chuột hỏi).
Bối cảnh xử lý task "Chạy một test tự động, nếu rớt (fail) thì hãy tự đi dò lỗi, sửa mã rồi push (đẩy) lên hệ thống." "Đoạn Query (truy vấn cơ sở dữ liệu) này tôi viết thiếu dấu phẩy ở đâu?"
Cửa sổ tiếp nhận
(Context Window)		 Bộ não suy tính quy trình Logic đỉnh cao (Reasoning). Không mỏi mệt khi phải tự đọc/viết file. Hỗ trợ Cửa Sổ Ngữ Cảnh cực khổng lồ, dư sức nuốt cả triệu dòng Code hiện hữu trong 1 giây.
Phần 4: Vũ khí sức mạnh

Chân dung AntiGravity (Phần 1: Tư duy)

1 Autonomous Agent

Khi giao cho AntiGravity một task khó lượng dữ liệu khổng lồ, nó không ngáoơ ra hỏi mà thực thi 100% bí mật ngầm:

  • Lập dàn ý các bước xử lý thuật toán (Planning).
  • Chạy hệ thống máy ảo ngầm (npx, git push command) mà không đòi hỏi ấn nút Run.
  • Tự bóc tách file project để đọc xuất dữ liệu.
  • Biết gọi công cụ tự vẽ sinh họa tiết nghệ thuật (Image Gen) để giao diện Web không bị trống Placeholder.

2 Kho tri thức KIs

Đây chính là "Bộ Não phân mảnh vĩnh cửu" (Knowledge Items):

  • Chúng được băm nhỏ và giấu trong phân vùng .gemini/antigravity/knowledge/ riêng tư cục bộ trên máy.
  • Lần sau vướng lỗi tương tự? Tự tìm KI cũ để vượt rào (bypass).
  • Tự phân cấp (Hierarchical Context): Giúp agent tự cảm nhận hiểu thứ tự cái gì quan trọng phải tôn trọng làm trước.

Phần 6: Khám Phá Kiến Trúc Tổ Hợp MCP

Khái niệm MCP Là gì?

"Hãy ví von: Giống như ổ cắm điện quốc tế cho Phích cắm AI gộp vào sức mạnh của thế giới Thực"

MCP (Model Context Protocol - Giao thức Ngữ cảnh Mô hình): Là một chuẩn giao tiếp thống nhất dạng mở (Open Standard - Tương tự như chuẩn Cáp viễn thông Type-C tiêu chuẩn thế giới) do Anthropic thiết kế gốc.

Nó phá bỏ lằn ranh giới hạn không gian mạng: Cho phép AI Agent tự biết đường "Cắm sạc liên kết" trực tiếp vào các kho ứng dụng thực tế ngoài đời của công ty bạn (Ví dụ: BigQuery SQL, Cổng Slack công ty, Kho Data Notion) một cách an toàn nhất.

Sức mạnh của MCP khi Team DA ứng dụng
