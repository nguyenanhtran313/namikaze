Phần 1: Kiến trúc tổng quan (Core Architecture)

1. Autonomous Agent Xử Lý Data Như Thế Nào?

Các hệ thống AI này hoạt động qua giao diện CLI (Command Line Interface). Thay vì dùng giao diện kéo-thả truyền thống (GUI), bạn dùng lệnh văn bản để yêu cầu AI tương tác trực tiếp với máy tính của mình.

Bản chất của loại Tool này là AI Tự Hành (Autonomous Agent). Chỉ với một yêu cầu (prompt), chúng sẽ tự động vạch kế hoạch, đọc file thư mục, tìm lỗi và tự viết code mà không cần bạn phải theo sát từng bước. Điểm khác biệt lớn nhất giữa chúng nằm ở khả năng nhớ trạng thái Database và Lưu Trí nhớ dài hạn (Memory Persistence).

*Ý là: Không cần quan tâm tới "CLI" hay "Lưu trạng thái Node". Hệ thống Agent sẽ âm thầm lo các việc xử lý kĩ thuật phía sau (background processing), Data Analyst chỉ việc cung cấp ngôn ngữ tự nhiên và theo dõi quá trình chạy Data.

*Node (Điểm nút): Node là một bước xử lý, một trạm dừng hoặc một phân nhánh trong chuỗi tư duy giải quyết vấn đề của AI.

*Lưu trạng thái (State): Là việc hệ thống có khả năng "ghi nhớ" toàn bộ bối cảnh, lỗi cũ, các biến số dữ liệu, và kết quả của các bước xử lý trước đó.

2. Bản chất cốt lõi của từng công cụ

Claude Code

Engine suy luận độc lập: Cực kỳ thông minh khi giải quyết các thuật toán nhiều bước. Tuy nhiên, AI này không lưu trạng thái (Stateless), tức là nó sẽ quên quy chuẩn của dự án trừ khi bạn đưa cho nó một quyển "sổ tay" quy tắc (file tĩnh CLAUDE.md).

Gemini CLI

Tương tác tức tốc (JIT Executor): Tối ưu cho tốc độ phản hồi cực nhanh. Tuy nhiên hệ thống thiếu đi bộ nhớ dài hạn, nó hiểu câu lệnh ngữ cảnh hiện tại rất tốt nhưng sẽ làm mới hoàn toàn trí nhớ khi bạn đóng cửa sổ Terminal.

AntiGravity

Hệ sinh thái kết hợp (Hybrid): Vừa có giao diện đẹp, vừa có sức mạnh truy cập hệ thống. Sức mạnh tuyệt đối nằm ở Bộ nhớ tự học (Knowledge Items - KIs), khả năng vẽ biểu đồ trực quan (Artifacts) và tự lướt Web ẩn (Browser Subagent) để lấy dữ liệu.

Phần 6: MCP - MODEL CONTEXT PROTOCOL LÀ GÌ

10. MCP (Model Context Protocol)

"MCP (Model Context Protocol — Giao thức Ngữ cảnh Mô hình) giống như ổ cắm điện chuẩn quốc tế. Nhà sản xuất thiết bị (quạt, tủ lạnh, laptop...) chỉ cần làm cái phích cắm đúng chuẩn — thì cắm vào ổ điện nước nào cũng chạy được. Không cần hỏi ổ điện đó của hãng nào."

Bản chất: MCP là chuẩn giao tiếp mở (open standard) do Anthropic tạo ra, cho phép AI Agent (tác tử AI) “cắm vào” bất kỳ công cụ/dịch vụ bên ngoài nào — miễn là công cụ đó “chìa ra” một MCP Server (máy chủ MCP).

Phần 7 — Module 1: Xoá Đệm Nhớ Hệ Thống

13. /compact

Bản chất kỹ thuật là Thu Gọn Số Lượng Token (Token Compaction), làm sạch bộ nhớ để mô hình không tốn công đọc các thông báo rác không quan trọng. Ý là:nén (tóm tắt) lịch sử hội thoại, tốn ít token hơn, AI bớt ngu. CLAUDE 2026 có tính năng AUTO-COMPACT nhưng bị chửi lắm, khuyên là hãy compact trước khi bị auto-compact.

1. Focus Compaction (Nén tái định hướng trọng số)

/compact Tập trung vào logic tính doanh thu theo tháng, bỏ qua các lỗi cài đặt môi trường vừa rồi

30 phút cãi nhau với AI để cài đặt một cái thư viện bị lỗi. Bây giờ lỗi đã xong, bạn muốn bắt đầu viết code chính, nhưng bộ não của AI đang bị "rác" bởi cả đống thông báo lỗi vớ vẩn nãy giờ => COMPACT. Kết quả: Bộ nhớ của nó bây giờ "sạch" và dồn 100% công lực vào việc tính toán.

2. Milestone Compaction (Nén chốt chặn hoàn thành)

/compact Chốt: Đã làm xong API đăng nhập và đăng ký. Bây giờ chuyển sang làm tính năng quên mật khẩu.

Mục đích: Xả bộ nhớ để máy chạy nhanh hơn nhưng vẫn giữ lại "kết quả cuối cùng" để làm bước tiếp theo. Kết quả: AI dọn dẹp toàn bộ code nháp, các câu hỏi phụ trong quá trình làm API. Nó chỉ ghi chú lại một dòng trong bộ nhớ: "API Login/Register: OK" => Giờ nó có rất nhiều khoảng trống để suy nghĩ cho tính năng "Quên mật khẩu".

💡 Nhận biết: Khi AI bị ngu, mắc chứng Ảo Giác Thuật Toán (Hallucination) do giới hạn Tokens Memory.

17. Listeners Hooks (Hàm Lắng Nghe Sự Kiện)

Gắn Script dọn dẹp Code và Bíp sau khi Agent chạy xong.

.claude/hooks.json
{
  "post-save": "sql-formatter --write",
  "post-run": "powershell -c \"([System.Media.SystemSounds]::Exclamation).Play()\""
}

"post-save" End-of-Stream, tự dưng mã Agent sinh SQL xong. Bạn chả cần làm gì, nền tảng bắn Process Pipeline (Terminal Script) kích chạy công cụ căn chỉnh Formats (sql-formatter). Trả ra Codebase vuông vức, sạch chuẩn mà chả tốn 10 giây Manual Human Interaction. Xong "post-run" nó làm kêu lên 1 tiếng "Bíp" để báo hiệu (Nhớ bật loa).
